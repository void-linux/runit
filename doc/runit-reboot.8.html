Name

runit-reboot - reboot the system

Synopsis

init 6
runit-reboot

Description

runit-reboot causes the Unix process no 1 runit(8) to reboot the system. Normally runit-reboot is run by runit-init(8) if called as init 6.

To signal runit(8) the system reboot request, runit-reboot touches the /etc/runit/reboot and /etc/runit/stopit files. Then a CONT signal is sent to runit(8).

Exit Codes

runit-reboot always returns 0.

See Also

runit-init(8), runit(8), runit-halt(8)
